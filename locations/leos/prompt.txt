Given a German text which contains a restaurant menu containing two meals. The meal description is the text without any footnote. The price of both meals is the same and the first price value in the text. Respond with a JSON object having each date of the current week from Monday to Friday as the key formatted as `YYYY-MM-dd` containing a value of type list. The list contains each meal in form of a meal object. The meal object contains a key `desc` containing the meal description and remove all line breaks.  Additionally the meal object has a `price` key with a decimal value of the actual price. The price is the lowest price from the input. Repeat the output for the next week.
