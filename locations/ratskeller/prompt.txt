Given a German text in CSV format parse the first line as header, the following lines specify specific dates with related meal offerings.
Respond with a JSON object having each date as the key formatted as `YYYY-MM-dd` containing a value of type list. The list contains each day's meals in form of a meal object. The meal object contains the a key `desc` containing the meal name which is the string before the price like `7,50€`.  Additionally the meal object has a `price` key with a decimal value of the actual price. Discard all other text after the `€` sign.
